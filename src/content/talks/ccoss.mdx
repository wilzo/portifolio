Durante minha formação como Tecnólogo em Sistemas para Internet, adquiri uma base sólida em desenvolvimento web, mobile, banco de dados, infraestrutura e boas práticas de programação. O curso foi voltado para preparar profissionais para atuar de forma prática na criação de soluções digitais modernas.

Aprendi desde os fundamentos da programação até conceitos avançados de desenvolvimento full stack, arquitetura de software, segurança da informação, e práticas ágeis de desenvolvimento.

Além disso, participei de projetos integradores, onde desenvolvemos sistemas completos aplicando todas as tecnologias estudadas.

**Tecnologias e ferramentas utilizadas durante o curso:**

- **Frontend:** HTML5, CSS3, JavaScript, TypeScript, React.js, Bootstrap, Tailwind CSS
- **Backend:** Node.js, Java, Python, Express.js
- **Banco de dados:** MySQL, PostgreSQL, SQLite
- **Infraestrutura e DevOps:** Git, GitHub, GitLab CI/CD
- **Ferramentas de desenvolvimento:** VS Code, Figma, Postman, Insomnia

Esse curso foi essencial para consolidar minha base como desenvolvedor FullStack e me preparar para desafios reais no mercado de tecnologia.
